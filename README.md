<h1>记账管家</h1><br style="color:red"/>

&emsp;闲暇时的一个小应用，记账管家，记录生活的每一笔收入开支。本项目使用Eclipse开发而成，编码采用GBK，下载导入时需要改一改eclipse编码，否则乱码，当然屏幕适配也有问题，测试用的4.7寸，但在5.1寸上显示比较尴尬咯。唯一的特色在于图标显示部分。
